# camanchaca-cms

Digital Signage Management System

## Quick Start

```bash
cd server
pnpm install
pnpm dev
```

Server starts on port 3001. First visitor becomes admin automatically.

## Structure

- `server/` — Node.js/Express backend
- `frontend/` — Vanilla JS admin dashboard
- `android/` — Android TV player (Kotlin)
- `scripts/` — Deployment utilities

## Requirements

- Node.js 20.6–22.x
- pnpm
